## 1. 角色说明

管理员拥有系统最高权限，可使用全部功能模块。系统共有 4 种角色：

| 角色 | 英文标识 | 说明 |
|------|----------|------|
| 管理员 | admin | 全部功能，含用户管理和系统配置 |
| 组织者 | organizer | 题库、试卷、考试、成绩、素材管理 |
| 审题员 | reviewer | 题目审核 |
| 被测者 | examinee | 参加考试、查看成绩 |

## 2. 用户管理

路径：左侧导航栏 → **用户管理**

### 2.1 用户列表

- 查看所有注册用户的用户名、邮箱、角色、状态
- 支持按角色筛选、关键词搜索

### 2.2 创建用户

1. 点击 **新建用户**
2. 填写用户名、邮箱、密码、选择角色
3. 点击 **确定** 完成创建

### 2.3 编辑用户

1. 在用户列表中点击目标用户的 **编辑** 按钮
2. 可修改用户名、邮箱、角色
3. 点击 **保存**

### 2.4 停用/启用用户

- 点击用户行的 **停用** 按钮可禁用该账号（软删除）
- 已停用账号可点击 **启用** 恢复

### 2.5 重置密码

- 点击 **重置密码**，系统将该用户密码重置为默认值
- 通知用户登录后自行修改

## 3. 系统配置

路径：左侧导航栏 → **系统配置**

### 3.1 LLM 服务商配置

系统支持多 LLM 服务商，每个服务商可独立配置：

| 配置项 | 说明 |
|--------|------|
| 服务商名称 | 自定义标识（如"本地 vLLM"、"DeepSeek"） |
| API Base URL | LLM 服务地址（如 `http://localhost:8100/v1`） |
| API Key | 认证密钥（本地 vLLM 可留空） |
| 模型名称 | 模型标识（如 `Qwen/Qwen3.5-35B-A3B`） |

> **优先级**：UI 配置 > `.env` 环境变量。在 UI 中配置后，`.env` 中的同名配置将被覆盖。

### 3.2 模块 LLM 分配

系统各功能模块可独立指定使用的 LLM 服务商：

| 模块 Key | 功能说明 |
|----------|----------|
| `question_generation` | AI 出题 |
| `scoring` | AI 评分 |
| `review` | AI 审核 |
| `interactive` | 情景互动 |
| `annotation` | 自动标注 |
| `intent` | 意图解析 |
| `indicator` | 指标分析 |
| `paper_generation` | 试卷生成 |
| `paper_import` | 试卷导入解析（新增） |

操作步骤：

1. 在系统配置页面找到 **模块 LLM 分配** 区域
2. 为每个模块选择对应的 LLM 服务商
3. 点击 **保存**
4. 配置即时生效，无需重启服务

## 4. LLM 状态监控

### 4.1 Dashboard 状态提示

登录后在 **Dashboard** 页面顶部可看到 LLM 状态横幅：

| 状态 | 显示 | 说明 |
|------|------|------|
| 正常 | 无横幅 | LLM 服务已连通 |
| 未配置 | 黄色警告横幅 | 未配置任何 LLM 服务商，点击"前往配置"跳转系统配置页 |
| 不可用 | 红色警告横幅 | LLM 已配置但连接失败 |

### 4.2 操作前检测

在以下操作执行前，系统会自动检测 LLM 状态并提示：

- AI 出题
- 试卷导入（LLM 解析）
- 交卷评分

如 LLM 不可用，系统会弹窗提示，部分功能可降级运行（如出题降级为模板生成）。

## 5. 题库管理

路径：左侧导航栏 → **题库管理**

### 5.1 浏览与搜索

- 按题型、难度、维度、标签筛选
- 支持关键词搜索
- 列表显示题型、难度、状态、创建时间

### 5.2 AI 辅助出题

1. 点击 **AI 出题**
2. 选择出题方式：
   - **基于素材**：选择已上传的素材，AI 根据素材内容生成题目
   - **自由主题**：描述出题需求（题型、知识点、难度等）
3. 设定题目数量、题型分布、Bloom 认知层次
4. AI 生成后进入预览界面，可逐题编辑或删减
5. 确认后保存到题库，题目进入草稿状态

### 5.3 手动创建题目

1. 点击 **新建题目**
2. 选择题型：单选题、多选题、判断题、填空题、简答题
3. 填写题干、选项（如适用）、正确答案
4. 设置难度等级（1-5）、所属维度、标签
5. 点击 **保存**

### 5.4 题目审核

1. 筛选状态为 **草稿** 的题目
2. 逐题查看或使用 AI 质量检查（五维度自动评分）
3. 点击 **通过** 或 **拒绝**
4. 支持批量审核：勾选多题后点击 **批量通过** / **批量拒绝**

### 5.5 批量删除

1. 在题目列表中勾选要删除的题目
2. 点击 **批量删除**
3. 确认弹窗后执行（系统会自动清理关联的试卷题目、考试题目、答案、评分明细等记录）

### 5.6 导入导出

- **Markdown 导入**：按规定格式编写题目文件后导入
- **Markdown 导出**：将题目导出为 Markdown 文件

## 6. 试卷管理

路径：左侧导航栏 → **试卷管理**

### 6.1 试卷列表

- 显示所有草稿和已发布试卷
- 列表包含标题、状态、题目数、总分、创建者、创建时间
- 已归档试卷不在此页面显示

### 6.2 创建试卷

1. 点击 **新建试卷**
2. 填写试卷标题、描述（可选）、考试时限（可选）
3. 点击 **确定** 创建草稿试卷

### 6.3 Word 导入试卷

1. 点击 **Word 导入**
2. 选择 `.docx` 文件
3. 系统执行两阶段解析：
   - **第一阶段**：正则解析识别大题结构、题型、分值、题目内容
   - **第二阶段**：LLM 智能验证修正题型和答案，生成置信度评分
4. 解析完成后进入 **交互式预览** 界面：
   - 卡片式展示每道题目（题干、选项、答案）
   - 三色状态标识：绿色（高置信度）、黄色（需注意）、红色（需修正）
   - 所有字段均可直接编辑
   - 可切换题型，答案控件自动适配
   - 可点击"采纳 AI 建议"接受单题修正，或"全部采纳"批量接受
5. 确认后点击 **导入**，系统调用 `import-reviewed` 端点完成入库

> **LLM 降级**：如 LLM 不可用，跳过第二阶段，直接展示正则解析结果。

### 6.4 编辑试卷

1. 点击试卷标题进入详情
2. 添加/删除大题，每个大题设置题型和分值规则
3. 从题库选题或调整已有题目
4. 拖拽排序、设置每题分值
5. 总分自动计算

### 6.5 发布试卷

1. 确认内容无误后点击 **发布**
2. 试卷状态变为"已发布"，内容锁定
3. 系统自动创建关联考试

### 6.6 归档试卷

1. 对已发布试卷点击 **归档**
2. 试卷从管理列表消失，关联考试自动关闭
3. 归档试卷在 **试卷归档** 页面可查看、恢复或永久删除

### 6.7 导出试卷

| 格式 | 说明 |
|------|------|
| Word (.docx) | 保留大题结构和分值信息 |
| JSON | 标准化格式，便于系统间数据交换 |

### 6.8 试卷题目同步至题库

1. 打开试卷详情，点击 **导入题库**
2. 系统比对每题状态（已入库/草稿/已修改/缺失）
3. 勾选需要导入的题目，点击 **执行导入**

## 7. 考试管理

路径：左侧导航栏 → **考试管理**

### 7.1 考试列表

- 考试由试卷发布时自动创建
- 列表显示考试标题、关联试卷、时限、参与人数、状态

### 7.2 考试详情

- 查看关联试卷信息
- 查看参加考试的考生列表
- 查看已提交的答卷和成绩

### 7.3 关闭考试

- 试卷归档时自动关闭关联考试
- 也可手动关闭考试，阻止新考生进入

---

## 8. 成绩管理

路径：左侧导航栏 → **成绩管理**

### 8.1 成绩列表

- 查看所有考生的考试成绩
- 按考试、考生筛选
- 支持排序（按总分、提交时间）

### 8.2 成绩详情

- 查看单个考生的逐题得分
- 查看 AI 评分详情（主观题评分理由）
- 查看五维诊断报告（AI 基础知识/技术应用/伦理安全/批判思维/创新实践）

### 8.3 成绩导出

- 支持导出为 Excel/CSV 格式
- 可导出班级/考试维度的汇总报告

### 8.4 评分申诉

- 查看考生提交的评分申诉
- 审核并处理申诉（维持/调整分数）

## 9. 素材管理

路径：左侧导航栏 → **素材管理**

### 9.1 上传素材

1. 点击 **上传** 按钮
2. 选择文件（支持 PDF、DOCX、TXT 格式）
3. 上传后系统自动解析为知识单元

### 9.2 素材使用

- 素材可在 AI 出题时引用
- 支持关键词搜索和筛选
