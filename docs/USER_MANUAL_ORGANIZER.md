## 1. 角色说明

组织者负责日常的出题、组卷、考试管理工作。与管理员相比，组织者 **不具备** 以下权限：

| 功能 | 是否可用 |
|------|----------|
| 用户管理 | 不可用 |
| 系统配置（LLM 服务商配置、模块分配） | 不可用 |
| 题库管理 | 可用 |
| 试卷管理 | 可用 |
| 考试管理 | 可用 |
| 成绩查看 | 可用 |
| 素材管理 | 可用 |

## 2. 题库管理

路径：左侧导航栏 → **题库管理**

### 2.1 浏览题库

- 按题型、难度、维度、标签筛选
- 支持关键词搜索

### 2.2 AI 辅助出题

1. 点击 **AI 出题**
2. 选择出题方式：
   - **基于素材**：选择已上传的素材，AI 根据内容生成题目
   - **自由主题**：描述出题需求（题型、知识点、难度等）
3. 设定题目数量、题型分布、Bloom 认知层次
4. AI 生成后进入预览界面，可逐题编辑或删减
5. 确认后保存到题库（草稿状态）

> **提示**：如 LLM 未配置或不可用，系统会弹窗提示。出题功能可降级为模板生成。

### 2.3 手动创建题目

1. 点击 **新建题目**
2. 选择题型：单选题、多选题、判断题、填空题、简答题
3. 填写题干、选项、正确答案
4. 设置难度、维度、标签
5. 保存（草稿状态）

### 2.4 题目审核

- 筛选草稿题目，逐题审核或批量审核
- 可使用 AI 质量检查辅助审核（五维度自动评分）
- 审核通过后题目可用于组卷

### 2.5 编辑题目

- 点击题目进入编辑页面
- 可修改题干、选项、答案、难度、标签等
- 已审核题目修改后会回到草稿状态

## 3. 试卷管理

路径：左侧导航栏 → **试卷管理**

### 3.1 试卷列表

- 显示草稿和已发布试卷
- 列表包含标题、状态、题目数、总分、创建者
- 支持搜索和筛选

### 3.2 创建试卷

1. 点击 **新建试卷**
2. 填写标题、描述（可选）、考试时限（可选）
3. 点击 **确定**

### 3.3 Word 导入试卷

1. 点击 **Word 导入**，选择 `.docx` 文件
2. 系统自动解析试卷结构（大题、题型、分值、题目内容）
3. 如 LLM 可用，系统还会对题型和答案进行智能验证
4. 进入 **交互式预览** 界面：
   - 卡片式展示每道题，全部字段可编辑
   - 三色状态标识（绿/黄/红）帮助快速定位问题
   - 可切换题型，答案控件自动适配
   - 支持"采纳 AI 建议"和"全部采纳"
5. 检查确认后点击 **导入** 完成

**支持的试卷格式**：

| 大题格式 | 示例 |
|----------|------|
| 中文序号 + 题型 | 一、判断题（每题2分，共20分） |
| "第X部分"格式 | 第一部分：单项选择题 |
| 括号序号格式 | （一）多项选择题 |
| 独立题型标题 | 简答题（每题10分） |

### 3.4 编辑试卷

1. 点击试卷标题进入详情
2. 操作：
   - 添加/删除大题
   - 从题库选题
   - 设置每题分值
   - 拖拽排序
3. 总分自动计算

### 3.5 发布试卷

1. 确认内容后点击 **发布**
2. 试卷锁定不可修改
3. 系统自动创建关联考试

### 3.6 归档与恢复

- 对已发布试卷点击 **归档**（关联考试自动关闭）
- 在 **试卷归档** 页面查看、恢复或永久删除

### 3.7 导出试卷

- 支持 Word (.docx) 和 JSON 两种格式

### 3.8 试卷题目同步至题库

1. 打开试卷详情，点击 **导入题库**
2. 系统比对每题状态
3. 勾选需要的题目，执行导入

## 4. 考试管理

路径：左侧导航栏 → **考试管理**

### 4.1 考试列表

- 考试由试卷发布时自动创建
- 列表显示考试标题、时限、参与人数、状态

### 4.2 关闭考试

- 试卷归档时考试自动关闭
- 也可手动关闭

---

## 5. 成绩查看

路径：左侧导航栏 → **成绩管理**

- 查看所有考生的成绩列表
- 查看单个考生的逐题得分和诊断报告
- 支持按考试、考生筛选

> **注意**：组织者可查看成绩但不处理评分申诉（申诉由管理员处理）。

## 6. 素材管理

路径：左侧导航栏 → **素材管理**

### 6.1 上传素材

1. 点击 **上传**
2. 选择文件（支持 PDF、DOCX、TXT）
3. 上传后自动解析为知识单元

### 6.2 使用素材

- 在 AI 出题时选择素材作为出题依据
- 支持搜索和筛选

## 7. 权限限制

以下功能仅管理员可用，组织者无法访问：

| 功能 | 说明 |
|------|------|
| 用户管理 | 创建/编辑/停用用户、分配角色 |
| 系统配置 | LLM 服务商配置、模块 LLM 分配 |
| 评分申诉处理 | 审核和处理考生的评分申诉 |

如需以上功能，请联系管理员。

## 8. 常见问题

### Q: AI 出题时提示"LLM 不可用"怎么办？

A: LLM 服务需要由管理员在系统配置中配置。出现此提示时，可联系管理员检查 LLM 配置，或使用降级的模板出题功能。

### Q: Word 导入后题型不正确？

A: 在交互式预览界面中，可以通过下拉菜单直接修改题型。建议同时关注三色状态标识，红色和黄色的题目需要重点检查。

### Q: 试卷发布后发现错误怎么办？

A: 已发布试卷无法修改。可以先归档该试卷，然后恢复为草稿状态重新编辑后再发布。

### Q: 导入题库时显示"已入库"是什么意思？

A: 说明该题目已在题库中且为已审核状态，无需重复导入。如果试卷中对该题有修改，会显示为"已修改"，可作为新题目导入。
