## 1. 系统简介

AI素养评测平台是一套面向教育场景的智能评测系统，支持：

- AI辅助出题与审核
- 试卷组装与管理
- 在线考试与自动评分
- 多维度素养指标分析

系统通过浏览器访问，推荐使用 Chrome、Edge 等现代浏览器。

## 2. 登录与注册

### 2.1 注册账号

1. 打开平台首页，点击 **注册**
2. 填写用户名、邮箱、密码
3. 点击 **提交注册**
4. 注册成功后自动跳转到登录页面

### 2.2 登录系统

1. 输入用户名和密码
2. 点击 **登录**
3. 登录成功后进入系统主页

## 3. 题库管理

### 3.1 浏览题库

- 左侧导航栏点击 **题库管理**
- 支持按题型、难度、维度、标签进行筛选
- 支持关键词搜索

### 3.2 创建题目

1. 点击 **新建题目**
2. 选择题型：单选题、多选题、判断题、填空题、简答题等
3. 填写题干、选项（如适用）、正确答案
4. 设置难度等级（1-5）、所属维度、标签
5. 点击 **保存**，题目进入草稿状态

### 3.3 AI辅助出题

1. 点击 **AI出题**
2. 描述出题需求（题型、知识点、难度等）
3. AI 自动生成题目，可在生成结果上进行修改
4. 确认后保存到题库

### 3.4 题目审核

- 草稿题目需经过审核才能用于正式试卷
- 支持批量审核操作
- 审核通过的题目状态变为 **已审核**

## 4. 试卷管理

### 4.1 试卷列表

- 左侧导航栏点击 **试卷管理**
- 列表显示所有 **草稿** 和 **已发布** 状态的试卷
- 已归档试卷不在此页面显示（见 4.8 节）
- 支持按状态筛选、关键词搜索

### 4.2 创建试卷

1. 点击 **新建试卷**
2. 填写试卷标题、描述（可选）、考试时限（可选）
3. 点击 **确定** 创建草稿试卷

### 4.3 编辑试卷结构

1. 在试卷列表点击试卷标题，进入详情
2. **添加大题**：点击 **添加大题**，输入大题标题（如"一、判断题"）
3. **添加题目**：在大题下点击 **从题库选题**，勾选需要的题目
4. **设置分值**：为每道题设置分值
5. **调整顺序**：通过拖拽或排序按钮调整大题和题目顺序
6. 总分会自动计算并显示

### 4.4 Word导入试卷

1. 在试卷列表页点击 **Word导入**
2. 选择 `.docx` 格式的试卷文件
3. 系统自动解析试卷结构：
   - 识别试卷标题
   - 识别大题分区和题型
   - 识别每题分值
   - 解析题目内容和选项
4. 导入完成后自动打开试卷详情
5. 可在详情中检查和调整解析结果

**支持的试卷格式**：

| 大题格式 | 示例 |
|----------|------|
| 中文序号 + 题型 | 一、判断题（每题2分，共20分） |
| "第X部分"格式 | 第一部分：单项选择题 |
| 括号序号格式 | （一）多项选择题 |
| 独立题型标题 | 简答题（每题10分） |

**支持的分值格式**：

| 格式 | 示例 |
|------|------|
| 每题X分 | 每题2分 |
| 每小题X分 | 每小题5分 |
| X分/题 | 4分/题 |
| 共X分 | 共30分 |
| 满分X分 | 满分20分 |

### 4.5 试卷题目同步至题库

试卷中的题目可以反向导入题库，实现试卷作为题目来源：

1. 打开试卷详情
2. 点击 **导入题库** 按钮
3. 系统自动比对每道题目的状态：
   - **已入库**：题目已在题库中且已审核，无需操作
   - **草稿**：题目在题库中为草稿状态，可提升为已审核
   - **已修改**：题目在试卷中有覆写修改，可作为新题目入库
   - **缺失**：题目不在题库中（通常不会出现）
4. 勾选需要导入的题目
5. 点击 **执行导入**

### 4.6 发布试卷

1. 确认试卷内容无误后，点击 **发布**
2. 试卷状态变为 **已发布**
3. 系统自动创建一场关联考试
4. 发布后试卷内容不可修改

### 4.7 导出试卷

- 支持导出为 **Word (.docx)** 格式
- 支持导出为 **JSON** 格式
- 在试卷详情中点击相应导出按钮

### 4.8 归档与恢复

**归档试卷**：
1. 在试卷列表中，对已发布试卷点击 **归档**
2. 试卷立即从试卷管理页面消失
3. 关联考试自动关闭

**管理归档试卷**：
1. 左侧导航栏点击 **试卷归档**
2. 查看所有已归档的试卷
3. 可执行 **恢复**（回到草稿状态）或 **永久删除**

## 5. 考试管理

### 5.1 考试列表

- 左侧导航栏点击 **考试管理**
- 考试由试卷发布时自动创建
- 列表显示所有考试及其基本信息

### 5.2 考试详情

- 点击考试标题查看详情
- 查看关联试卷、考试时间、参与人数等信息

### 5.3 考试生命周期

考试状态由关联试卷自动管理：
- 试卷发布 → 自动创建考试
- 试卷归档 → 自动关闭考试

## 6. 素材管理

### 6.1 上传素材

1. 左侧导航栏点击 **素材管理**
2. 点击 **上传** 按钮
3. 选择文件上传（支持文档、图片等格式）
4. 上传完成后素材自动解析并入库

### 6.2 素材使用

- 素材可在出题、组卷等场景中引用
- 支持素材关键词搜索

## 7. 常见问题

### Q: Word导入后题目类型不正确？

A: 请确保试卷中每个大题的标题包含明确的题型关键词（如"判断题"、"选择题"、"简答题"等）。系统通过标题中的关键词识别题型。

### Q: Word导入后分值不正确？

A: 请确保大题标题或其后一行包含分值描述（如"每题2分"或"共20分"）。如果仅有总分描述，系统会自动按题目数量平均分配。

### Q: 导入题库时显示"已入库"怎么办？

A: 这说明该题目已经在题库中且为已审核状态，无需重复导入。如果试卷中对该题目有修改（覆写），会显示为"已修改"状态，此时可以作为新题目导入。

### Q: 归档的试卷还能恢复吗？

A: 可以。进入 **试卷归档** 页面，找到对应试卷点击 **恢复** 即可。恢复后试卷回到草稿状态，需重新发布。

### Q: 试卷总分不是100分正常吗？

A: 系统不强制总分为100分，但大多数正式试卷总分为100分。如果导入后总分异常，建议检查分值设置是否合理。
